# Netflix Content Strategy Analysis 📊

## Overview

This project analyzes Netflix's content catalog to understand content patterns, audience targeting, and content strategy using exploratory data analysis (EDA).

The goal is to extract meaningful insights from Netflix's movies and TV shows dataset and provide data-driven recommendations.

---

## Dataset

Dataset: Netflix Movies and TV Shows Dataset

The dataset contains 8,807 Netflix titles with information including:

- Content type
- Country
- Release year
- Rating
- Duration
- Genres

---

## Tools Used

- Python
- Pandas
- Matplotlib
- Jupyter Notebook

---

## Analysis Performed

### Content Type Analysis
Compared Movies and TV Shows to understand Netflix's catalog distribution.

### Release Year Analysis
Analyzed release trends to understand the age distribution of Netflix content.

### Genre Analysis
Identified the most common genres available on Netflix.

### Audience Rating Analysis
Explored content ratings to understand target audience segments.

### Country Analysis
Analyzed major content-producing regions.

### Duration Analysis
Studied movie length distribution patterns.

---

## Key Insights

- Netflix's catalog contains more Movies compared to TV Shows.
- The platform's catalog mainly consists of modern content.
- International Movies, Dramas, and Comedies dominate Netflix's genres.
- Mature audience content represents a major portion of the catalog.
- The United States contributes the largest amount of content.
- Most movies follow standard feature-length durations.

---

## Recommendations

- Increase variety by expanding family-friendly content categories.
- Include more classic titles to improve content diversity.
- Continue investing in international content to maintain global appeal.
